# cnweb-wc26-resource

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖各类通用网页资源。

## 项目简介

这是一个静态资源集合仓库，主要存放多个独立制作的 HTML 页面。这些页面以归档形式存在，可用于参考、学习或直接使用。仓库本身不针对任何特定网站或域名，仅作为网页素材的集中存储点。

## 目录结构

```
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- `pages/`：每个子目录或文件对应一个独立 HTML 页面，页面之间无依赖关系。
- `assets/`：存放页面可能引用的通用样式、脚本或图片资源，部分页面可能自带内联资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面可能包含样式、脚本及少量外部资源引用，但尽量保持自包含。
- 归档内容不涉及任何具体商业或推广信息，仅供技术参考。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/cnweb-wc26-resource.git
   ```
2. 打开 `pages/` 目录下的任意 HTML 文件即可查看页面效果。
3. 如需部署到服务器，可直接将整个仓库内容上传至静态托管服务。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 commit 记录。
- 欢迎提交 Issue 或 Pull Request 提供改进建议。
